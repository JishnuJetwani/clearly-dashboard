## Clearly 

## What It Does

This project automates the onboarding and referral verification process end-to-end.

- Provides a dashboard displaying all candidates and their current onboarding stage
- Sends automated referral emails with secure, tokenized links
- Uses an AI voice agent to call and interview referrals
- Summarizes call transcripts with Gemini AI and evaluates candidate suitability
- Sends a final decision email based on the referral outcome
